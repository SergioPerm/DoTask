# Tasker - personal task manager
- add and check tasks
- note the importance in task 
- add reminder

used frameworks:
- Core Data
- Core Animation
- Core Graphics
- UIKit
- User Notifications

Architectural patterns:
- MVVM + Coordinator, MVC

iOS 11.* support
